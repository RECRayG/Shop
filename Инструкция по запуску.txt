1) Версия Java: 18.0.2.1
2) Система сборки: Maven (v 3.6.3)
3) PostgreSQL v16 (необходимо развернуть базу данных с именем shop, пользователь postgres и пароль 12345)
4) Для запуска программы необходимо прописать следующую команду из директории проекта:
	java -jar Shop.jar search input.json output.json
	или
	java -jar Shop.jar stat input.json output.json
	, где input.json - входной файл, output.json - выходной файл

P.S. Информация выводиться в файл в одну строку, не успел отформатировать вывод в файл, но всё ТЗ соблюдено
